🧠 Python Class Notes (Interactive + Explained)

  • Dar in jalase ba Spyder va Jupyter Notebook ashna shodim.
  • Spyder yek IDE hast baraye Python ke barname nevisi ro asan mikone.
  • Jupyter Notebook be sorat celli code mizari va ejra mikoni. Baraye data science fogholade ast.
  • Kar jadidi anjam nashod. Jalase badi class shoru mishavad.

  • 📝 Motaghayer (variable) yek esm baraye zakhire data ast: 
    
x = 5
user_score = 90
  • 🐍 Estefade az snake_case baraye esm gozari: 
    
my_variable_name = "Hello"
  • 🖥️ Ejra kardan code ha dar Python be tore khati va radifi ast.
  • 📢 Baraye chap kardan dar Python az print() estefade mikonim. \n baraye khat jadid ast: 
    
print("Hello\nWorld")  # output:
# Hello
# World
  • 🔎 Baraye moshakhas kardan noe motaghayer az type() estefade konid: 
    
a = 5
print(type(a))  # output: <class 'int'>

b = 3.14
print(type(b))  # output: <class 'float'>
  • ➗ Fargh / va // dar Python: 
    
print(7 / 3)   # output: 2.3333333333333335 (decimal division)
print(7 // 3)  # output: 2 (integer division, floor)
  • 🔄 Taghyir noe motaghayer: 
    
b = 5
b = float(b)  # taghir be float
print(type(b))  # output: <class 'float'>

a = 3.7
a = int(a)    # taghir be int (ghat konande)
print(a)      # output: 3
  • ✖️ Zarb dar khod ba estefade az operator haye mukhtalef: 
    
a = 4
a *= 4  # a = a * 4
print(a)  # output: 16

  • ➕ Jam kardan string ha ba +: 
    
first = "Hello"
second = "World"
print(first + " " + second)  # output: Hello World
  • 🎯 Dastresi be harf khas: 
    
name = "Ali"
print(name[0])  # output: A
  • 📏 Tool string: 
    
text = "Hello123"
print(len(text))  # output: 8
  • ✂️ Boresh (slice): 
    
text = "Python"
print(text[1:4])  # output: yth
print(text[::2])  # output: Pto
  • 🔁 Taghir kalame: 
    
text = "Hello Ali"
print(text.replace("Ali", "Reza"))  # output: Hello Reza
  • 🔍 Jostojo dakhel string: 
    
text = "welcome to python"
print(text.find("python"))  # output: 11
print(text.find("java"))    # output: -1
  • 🔠 Lowercase kardan: 
    
text = "HELLO World 123!"
print(text.lower())  # output: hello world 123!
  • 🔠 Uppercase kardan: 
    
text = "hello world"
print(text.upper())  # output: HELLO WORLD
  • 🔢 Count method: 
    
text = "banana"
print(text.count("a"))  # output: 3

  • 📋 (list) baraye list kardan item ha estefade mishe.
  • ➕ (append) baraye ezafe kardan item be list estefade mishe (eyn item be enteha ezafe mishe). 
    
fruits = ["apple"]
fruits.append("banana")
print(fruits)  # ['apple', 'banana']
  • ➕➕ (extend) mitone chand item ro be list ezafe kone (mitone list dige ro ham ezafe kone). 
    
fruits = ["apple"]
fruits.extend(["banana", "orange"])
print(fruits)  # ['apple', 'banana', 'orange']
  • 🎯 (insert) baraye ezafe kardan item dar position khas estefade mishe. 
    
fruits = ["apple", "orange"]
fruits.insert(1, "banana")
print(fruits)  # ['apple', 'banana', 'orange']
  • ❌ (remove) baraye pak kardan item mokhtas az list estefade mishe. 
    
fruits = ["apple", "banana"]
fruits.remove("apple")
print(fruits)  # ['banana']
  • ❌ (pop) ham baraye pak kardan item az list estefade mishe (az enteha ya index khas). 
    
fruits = ["apple", "banana"]
fruits.pop()
print(fruits)  # ['apple']
  • 🧹 (clear) tamam item ha ro az list pak mikone. 
    
fruits = ["apple", "banana"]
fruits.clear()
print(fruits)  # []
  • 🔍 (index) baraye peyda kardan position avalin bar az item estefade mishe. 
    
fruits = ["apple", "banana", "apple"]
print(fruits.index("apple"))  # 0
  • 🔁 (reverse) baraye makoos kardan list estefade mishe. 
    
fruits = ["apple", "banana"]
fruits.reverse()
print(fruits)  # ['banana', 'apple']
  • 🔢 (sort) baraye moratab kardan list estefade mishe. 
    
numbers = [5, 2, 9, 1]
numbers.sort()
print(numbers)  # [1, 2, 5, 9]

  • 📦 Tuple dar Python ye noe sakhtare dade hast ke mesle list ast vali chand tafavot dare, mesle amniyat bala va ghabele taghir nist. 
    
my_tuple = (1, 2, 3)
print(my_tuple[0])  # output: 1
# Tuple ha ghabele taghir nistand (immutable)
  • 🔢 count() dar tuple mesle list kar mikone, tedad baramad har item ro mide: 
    
t = (1, 2, 2, 3)
print(t.count(2))  # output: 2
  • ✂️ split() metod e ke ye string ro be ghesmat-haye kuchiktar (list) tabdil mikone bar asas ye karakter ya fasele: 
    
text = "hello world python"
words = text.split(" ")  # ya faghat split() ham mishe
print(words)  # output: ['hello', 'world', 'python']
  • 🪟 partition() ham mesle split hast vali faghat 3 ghesmat mide: ghesmate ghabli, separator, ghesmate akhar. 
    
text = "hello-world-python"
parts = text.partition("-")
print(parts)  # output: ('hello', '-', 'world-python')
  • 🔗 join() ye list az string-ha ro migire va ba ye karakter (ya string) vasat be yek string vasl mikone: 
    
words = ['hello', 'world', 'python']
joined = " ".join(words)
print(joined)  # output: hello world python
  • 🔢 range() ye tool baraye sakhtane sequence az adad ast ke bishtar too loop ha estefade mishe: 
    
for i in range(5):
    print(i)
# output: 0 1 2 3 4

  • 📦 Set ye noe data structure hast ke mitone chand value ro negah dare vali ba chand fargh mohem nesbat be list: - meghdar ha tekrari nadaran - tartib nadare (unordered) 
    
my_set = {1, 2, 3}
print(my_set)  # {1, 2, 3}
  • ➕ add(): baraye ezafe kardan ye item be set estefade mishe. 
    
my_set = {1, 2}
my_set.add(3)
print(my_set)  # {1, 2, 3}
  • 🔗 union(): do ya bishtar set ro be ham vasl mikone va set jadidi mide ke tamam item ha ro bedoon tekrar dare. 
    
a = {1, 2}
b = {2, 3}
print(a.union(b))  # {1, 2, 3}
  • ➕➕ update(): chand item ro az set, list ya tuple be set ezafe mikone. farghesh ba add() ine ke: - add() → faghat 1 item - update() → chand item 
    
a = {1}
a.update([2, 3])
print(a)  # {1, 2, 3}
  • 🎯 intersection(): meghdar haye moshtarak beyn set ha ro mide. 
    
a = {1, 2, 3}
b = {2, 3, 4}
print(a.intersection(b))  # {2, 3}
  • ➖ difference(): item haei ke faghat to set aval hastan va to set dovom nistand ro mide. 
    
a = {1, 2, 3}
b = {2, 4}
print(a.difference(b))  # {1, 3}
  • ❌ discard(): ye item khas ro hazf mikone, vali agar nabashe error nemide. 
    
a = {1, 2}
a.discard(2)
a.discard(5)  # no error
print(a)  # {1}
  • ❌ pop(): ye item ro be surat random az set hazf mikone. 
    
a = {10, 20, 30}
a.pop()
print(a)  # item tasadofi hazf shode
  • ❌ remove(): ye item khas ro hazf mikone, vali agar nabashe error mide (KeyError). 
    
a = {1, 2}
a.remove(1)
# a.remove(5)  # KeyError
  • 🔎 issubset(): check mikone aya hame item haye ye set to set digari hast ya na. 
    
a = {1, 2}
b = {1, 2, 3}
print(a.issubset(b))  # True
  • 📄 copy(): yek copy az set misaze ke mostaghel az set asli hast. 
    
a = {1, 2}
b = a.copy()
b.add(3)
print(a)  # {1, 2}
print(b)  # {1, 2, 3}
  • 🧾 Dictionary: ye data structure hast ke az key-value pair estefade mikone. 
    
person = {"name": "Ali", "age": 20}
print(person["name"])  # Ali
  • 🔧 dict(): yek function hast baraye sakhte dictionary. 
    
person = dict(name="Reza", age=25)
print(person["age"])  # 25
  • 🔍 get(): baraye gereftan value az dictionary, ba emkane meghdar pishfarz. 
    
person = {"name": "Sara"}
print(person.get("age", "Not Found"))  # Not Found
  • ❌ del: keyword hast baraye hazf kardan item az dictionary. 
    
person = {"name": "Ali", "age": 30}
del person["age"]
print(person)  # {"name": "Ali"}
  • ❌ pop() (dar dictionary): value ro barmigardoone va key ro hazf mikone. 
    
person = {"name": "Ali", "age": 22}
age = person.pop("age")
print(age)  # 22
print(person)  # {"name": "Ali"}
  • ❌ popitem(): akharin key-value ro hazf mikone va barmigardoone (az Python 3.7 be bad). 
    
person = {"name": "Ali", "city": "Tehran"}
print(person.popitem())  # ('city', 'Tehran')
